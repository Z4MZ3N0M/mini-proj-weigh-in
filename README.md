# 🛡️ WeighIn Tracker | Sensia AI

A premium, cinematically-themed fitness dashboard designed for the disciplined. Track your strength and log your progress with zero friction and high-end visual aesthetics.

![Warrior Dashboard](https://wallpapercave.com/wp/wp10528224.jpg)

## 🏆 Key Features
- **Cinematic Experience**: 60-second Ken Burns-style background rotations with high-resolution motivational imagery (Tom Hardy, Hajime no Ippo, Batman).
- **Premium UI**: Glassmorphism cards, custom spotlight effects, and specialized typography (**Bebas Neue** & **Space Grotesk**).
- **Full-Stack Logic**: Express.js backend with real-time CRUD operations synced to a cloud database.
- **Precision Stats**: Real-time calculation of Average Calories, Total Logs, and Latest Weight with UTC date accuracy.
- **Ready for Cloud**: Built-in support for **Supabase (Postgres)** and **Render** deployment.

## 🛠️ Tech Stack
- **Frontend**: Vanilla HTML5, CSS3, JavaScript.
- **Backend**: Node.js, Express.
- **Database**: PostgreSQL (Supabase).
- **Hosting**: Render (Web Service).

## 🚀 Local Setup
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Z4MZ3N0M/mini-proj-weigh-in.git
   cd mini-proj-weigh-in
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Configure Database**:
   - Set up a Postgres database (locally or on Supabase).
   - Add your connection string to a `.env` file or export it as `DATABASE_URL`.
4. **Start the server**:
   ```bash
   npm start
   ```
5. **Visit**: `http://localhost:3000`

## 🔑 Environment Variables
| Key | Value |
| :--- | :--- |
| `DATABASE_URL` | Your Postgres connection string (e.g., `postgresql://...`) |
| `PORT` | (Optional) Port number (defaults to 3000 or 10000 on Render) |
